# Family Bank — Interactive Prototype

A clickable mobile web prototype built for the PoliMi *Usability of Digital Systems* course.
Single self-contained `index.html` — no build step, no dependencies.

## Flows
1. **Single Transfer** — beneficiary → account & type → review & confirm
2. **Transactions** — list → filters → transaction details
3. **Spending** — overview → income & expenses / spending categories → transport detail → budget & alert

Navigation is unified under a Home hub and bottom tab bar, with smooth screen transitions and micro-animations.

## Run locally
Just open `index.html` in a browser. (Or serve the folder with any static server.)

## Live site
Hosted on GitHub Pages — see the repository's **Settings → Pages**.
